# Netlify Homepage Clone

A frontend clone of the Netlify homepage built using HTML, CSS, and JavaScript.

This project recreates key sections of the original site, including the navigation bar with mega dropdown menus, responsive hero layout, feature sections, and an interactive AI Gateway section with copy-to-clipboard functionality.

## Features

- Responsive layout (desktop, tablet, mobile)
- Mega dropdown navigation
- Tabbed content section
- Copyable code block
- Smooth hover animations

## Tech Stack

- HTML
- CSS (Flexbox & Grid)
- Vanilla JavaScript
